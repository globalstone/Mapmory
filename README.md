# 프로젝트 소개
일기 작성은 감정 정리와 자기 성찰에 도움이 됩니다. 전통적인 종이 일기 방식의 불편함을 해소하기 위해, 스마트폰과 웹 기술을 활용한 리마인드 앱 'Mapmory'를 고안하였습니다. 이 앱에서는 텍스트 뿐만 아니라 사진, 영상, 위치 정보를 기록할 수 있으며, 내부 SNS 서비스를 통해 다른 사람들과 자신의 생각과 감정 및 관련 정보를 공유할 수 있습니다. 일기의 개념을 확장하고 새로운 커뮤니케이션을 촉진하는 WEB 2.0 기반 플랫폼을 체험해보세요!

# 2. 시스템

## 1) 지도 시스템
- 사용자가 특정 위치에서 marker 형식으로 timeline을 기록
- marker를 click하면 간단한 text와 img 정보만 노출되며, 세부사항 click 시 동영상이나, 추가적인 hypermedia data까지 전부 조회 가능
- kakao map이나 naver map에서 기본으로 제공하는 marker/ private marker / public marker 총 세 가지의 서로 다른 marker 제공
- 각 marker는 Map 화면에서 버튼을 눌러 원하는 maker만 볼 수 있도록 filtering 기능 제공

## 2) 기록관리 시스템
- 사용자가 기록할 때 "timeline, 음악, 영상 및 사진, text"를 모두 기록 가능
- 기록은 실시간으로 기록 할 수 있으며, 현재 위치 정보만 기록한 후 여가 시간에 구체적으로 기록하는 기능
- 작성의 편의성을 위해, 사용자가 음성으로 기록한 내용을 텍스트로 변환하는 기능 제공
- keyword를 입력하여 내가 기록한 것들을 list 형태로 보여주는 기능 제공
- 임시 저장 기능 지원
- 늦게 오는 편지 (time capsule service)

## 3) 회상 시스템
- 이전에 작성했던 기록들을 정기적으로(오늘의 1년 전 등) notice하여  remind할 수 있는 기능 제공.
- keyword 및 hashtag 를 입력하여 내가 기록한 것들을 list 형태로 보여주는 기능.  (날짜 별 ,기간 별)
- keyword : 사용자가 작성한 기록들을 기반으로 추출된 단어들
- hashtag : 사용자가 '#'을 사용하여 즉흥적으로 표시한 단어들
- 최신 글 작성 이후, 해당 글에서 추출한 keyword와 관련된 과거의 글들을 자동으로 remind

## 4) 카테고리 시스템
- 사람들이 흔히 접할 수 있는 기본 category 제공
- 사용자가 category를 직접 세부 category를 customize할 수 있는 기능 제공
## 5) 추천 시스템
- AI를 이용해서 기록된 내용들에 대해 자동으로 생성된 주요 keyword를 생성. 이를 기반으로 사용자가 가볼만한 곳을 추천하는 기능 제공
- 사용자가 작성한 HashTag를 이용하여 추천기능.
- 공원, 온/오프라인 store, activity, 산책로

## 6) 회원 시스템
- 다양한 소셜 로그인 제공 ( 카카오,네이버,구글 등)
- 개인의 성별, 나이 등 기본 정보 회원 가입 단계에서 개인 정보 수집.

## 7) 고객센터 시스템
- 챗봇을 이용한 AI 고객센터 구축

## 8) 커뮤니티 시스템
- public / private 구분하여 원하는 기능만 공유할 수 있도록 함 (reaction, 이모지나 댓글 기능을 통해 소통 시스템 구축)
- follow 기능 제공
- hashtag를 기반 검색 system 제공

## 9) 결제 시스템
- 장소 및 주문 예약 기능 제공
- 선결제 기능 제공

## 10) 보안 시스템
- Service 및 서버 보안 logic 구현
